# Solo Leveling Daily System

A gamified productivity app inspired by the Solo Leveling universe. Complete real life goals as quests, earn XP, level up, and grow stronger every day.

**Live:** [vector94.github.io/solo-leveling](https://vector94.github.io/solo-leveling/)

---

## What it does

You play as a Hunter. Every task you complete in real life is a quest. Finish quests to earn XP and level up. Miss them and face a penalty. Just like in the System.

---

## Features

- Create daily, one time, and boss quests with difficulty ranks from E to A
- Earn XP on completion and level up with an ARISE modal
- Allocate stat points across STR, AGI, INT, SEN and VIT on every level up
- Rank up from E all the way to S based on your level
- Daily quests reset automatically at midnight and unfinished ones apply an XP penalty
- Track your daily streak for consecutive active days
- Unlock 15 achievements as you progress
- All data is saved locally so no account is needed
- Works on both desktop and mobile

---

## Stack

React, TypeScript, Vite, Tailwind CSS, Zustand, Framer Motion

## Run locally

```bash
npm install
npm run dev
```
